В i3 почти всё управление завязано на горячие клавиши. По умолчанию используется клавиша Mod (обычно это Win/Super, но может быть Alt — зависит от конфигурации).

  

Вот список основных комбинаций:

  

  

Базовые

  

  

- Mod+Enter — открыть терминал.
- Mod+d — dmenu (запуск приложений).
- Mod+Shift+q — закрыть окно.
- Mod+Shift+e — выйти из i3.
- Mod+r — режим изменения размеров окон (стрелками или hjkl).

  

  

  

Навигация

  

  

- Mod+стрелки или Mod+h/j/k/l — перемещение по окнам.
- Mod+Shift+стрелки — переместить окно в другую сторону.
- Mod+число (1–9) — переключиться на рабочее пространство.
- Mod+Shift+число — переместить окно на другое рабочее пространство.

  

  

  

Макеты окон

  

  

- Mod+e — расстановка по умолчанию (splits).
- Mod+s — табы (stacked).
- Mod+w — табы в ряд (tabbed).
- Mod+f — полноэкранный режим.

  

  

  

Разделение окон

  

  

- Mod+v — вертикальное разделение.
- Mod+h — горизонтальное разделение.

  

  

  

Прочее

  

  

- Mod+Shift+r — перезапуск i3 (без выхода из сессии).
- Mod+Shift+c — перечитать конфиг.
