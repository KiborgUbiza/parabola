---

📌 Установка Debian рядом с Windows (Dual Boot)

🧰 Что понадобится:

Флешка от 4 ГБ

Образ Debian ISO: https://www.debian.org/distrib/

Rufus для записи образа

Резервная копия важных файлов



---

🔧 Шаг 1: Подготовка места на диске (в Windows)

Win + X → Управление дисками

Сжать основной диск (например, C:)

Освободить минимум 20–30 ГБ

Оставить незанятое пространство



---

💽 Шаг 2: Создание загрузочной флешки

Запустить Rufus

Настроить:

Образ: Debian ISO

Схема раздела:

GPT (если Windows UEFI)

MBR (если BIOS)



Нажать Старт → дождаться окончания



---

🧭 Шаг 3: Настройка BIOS/UEFI

Зайти в BIOS (обычно Del, F2, Esc)

Включить UEFI (если Windows UEFI)

Выключить Secure Boot

Установить загрузку с флешки



---

🧱 Шаг 4: Установка Debian

Загрузиться с флешки → выбрать Graphical install

Установить язык, сеть и прочее

Разметка диска: Manual (ручная)

Найти свободное место и создать разделы:

/ (корень): 20–30 ГБ, ext4

swap: 2–4 ГБ

/home: остальное (по желанию)




---

🔁 Шаг 5: Установка GRUB

Согласиться на установку GRUB

Установить его в основной диск (например, /dev/sda)

GRUB должен автоматически найти Windows



---

✅ Шаг 6: Перезагрузка

Извлечь флешку

Перезагрузиться

В меню GRUB выбрать Debian или Windows



---

🆘 Если Windows не в списке GRUB

sudo update-grub
