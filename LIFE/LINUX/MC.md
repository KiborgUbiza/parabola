Midnight Commander (MC) — это мощный текстовый файловый менеджер для Linux, который работает в терминале. Он удобен для управления файлами, особенно на сервере или в минимальных окружениях. Вот инструкция по его использованию:
1. Установка MC

Для большинства дистрибутивов Linux Midnight Commander доступен в стандартных репозиториях:

Arch Linux:

sudo pacman -S mc

После установки запустите Midnight Commander командой:

mc

2. Основной интерфейс

Когда вы откроете MC, вы увидите два окна, которые отображают содержимое директорий (панели):
• Левая панель.
• Правая панель.

Навигация осуществляется с помощью клавиатуры, но можно подключить мышь.

3. Основные клавиши управления
Навигация:
• Стрелки: перемещение по файлам и папкам.
• Enter: открытие файла или переход в папку.
• Tab: переключение между панелями

Основные действия:
• F3: просмотр файла (режим “только чтение”).
• F4: редактирование файла.
• F5: копирование файлов.
• F6: перемещение/переименование файлов.
• F7: создание новой папки.
• F8: удаление файла/папки.
• F9: открытие верхнего меню.

Поиск и команды:
• Alt + ?: поиск файлов.
• Ctrl + O: скрыть/показать MC и вернуться в терминал.
  
4. Работа с архивами

Midnight Commander умеет открывать архивы как папки. Просто выделите архив, нажмите Enter, и MC откроет его содержимое.


5. Редактор файлов (F4)
MC использует встроенный текстовый редактор, который удобен для правки конфигураций или небольших скриптов.
• F2: сохранить изменения.
• F10: выйти из редактора.

6. Работа с удаленными системами

MC поддерживает подключение по FTP или SSH.

Пример подключения через SSH:

1. Нажмите Left/Right Panel → Shell link (или используйте комбинацию Ctrl+X, S).

2. Введите путь в формате:
/#sh:user@host

  
3. После ввода пароля вы сможете работать с удаленной системой, как с локальной папкой.

  7. Полезные настройки

  Для изменения настроек MC:

• Нажмите F9 → выберите Options → Configuration.
• Вы можете включить/отключить отображение скрытых файлов, подсветку, поддержку мыши и многое другое.

  8. Выход из MC
Для выхода нажмите F10.
