Продвинутый поиск в Linux

В Linux поиск файлов и информации обычно выполняется тремя основными инструментами:

- "find" — поиск файлов по параметрам
- "grep" — поиск текста внутри файлов
- "locate" — быстрый поиск по индексированной базе

---

1. find — мощный поиск файлов

Поиск файла по имени:

find /home -name "file.txt"

Поиск без учёта регистра:

find /home -iname "file.txt"

Поиск по расширению:

find /home -name "*.png"

---

Поиск по размеру

Файлы больше 100 MB:

find / -size +100M

Файлы меньше 1 MB:

find / -size -1M

---

Поиск по времени изменения

Файлы изменённые за последние 2 дня:

find /home -mtime -2

Файлы старше 30 дней:

find /home -mtime +30

---

Поиск по владельцу

find /home -user username

---

Поиск по правам доступа

Например файлы с правами "777":

find / -perm 777

---

2. grep — поиск текста внутри файлов

Поиск строки в файле:

grep "password" file.txt

Поиск во всей директории:

grep -r "password" /etc

Без учёта регистра:

grep -ri "password" /etc

Показать номер строки:

grep -rn "password" /etc

---

3. Комбинация find + grep

Поиск текста внутри определённых типов файлов.

Например поиск слова "port" во всех ".conf" файлах:

find /etc -name "*.conf" -exec grep "port" {} \;

---

4. locate — быстрый поиск файлов

Поиск файла:

locate firefox

Обновление базы файлов:

sudo updatedb

"locate" работает очень быстро, потому что ищет в предварительно созданной базе файлов.

---

5. fd — современная альтернатива find

Более простой и быстрый инструмент поиска.

Пример:

fd config

Поиск по расширению:

fd ".png"

---

6. Поиск больших файлов

Найти 20 самых больших файлов в системе:

du -ah / | sort -rh | head -20

---

Полезная формула поиска в Linux

Большинство задач решаются тремя командами:

- "find" — найти файл
- "grep" — найти текст
- "du" — узнать что занимает место
