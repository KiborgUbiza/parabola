Vim — это мощный текстовый редактор, основанный на Vi, который предоставляет обширные возможности для редактирования текста через клавиатурные команды. Он поддерживает множество функций, таких как подсветка синтаксиса, режимы редактирования и возможность работы с несколькими файлами одновременно.

Установка Vim в Arch Linux
 1. Установите Vim через пакетный менеджер pacman:

sudo pacman -S vim


 2. Проверьте, что Vim установлен:

vim --version

Основные режимы в Vim

Vim использует несколько режимов для редактирования текста:
 • Нормальный режим (Normal Mode) — используется для перемещения по тексту и выполнения команд.
 • Режим вставки (Insert Mode) — для ввода текста.
 • Режим визуального выделения (Visual Mode) — для выделения текста.
 • Командный режим (Command Mode) — для ввода команд (например, сохранение файла, выход и т.д.).

Переключение между режимами
 • В нормальном режиме нажмите i, чтобы перейти в режим вставки.
 • В режиме вставки нажмите Esc, чтобы вернуться в нормальный режим.
 • В нормальном режиме нажмите v или V для входа в визуальный режим.

Основные команды в Vim

Нормальный режим:
 • Перемещение по файлу:
 • h — влево (на один символ).
 • j — вниз (на одну строку).
 • k — вверх (на одну строку).
 • l — вправо (на один символ).
 • gg — перейти в начало файла.
 • G — перейти в конец файла.
 • w — перейти к следующему слову.
 • b — вернуться на одно слово назад.
 • Редактирование текста:
 • x — удалить символ под курсором.
 • dd — удалить текущую строку.
 • d + движение (например, dw для удаления слова).
 • u — отменить последнее действие.
 • Ctrl + r — повторить отменённое действие.
 • Поиск:
 • /текст — найти слово “текст” в файле.
 • n — перейти к следующему вхождению.
 • N — перейти к предыдущему вхождению.
 • Копирование и вставка:
 • y — копировать (yank).
 • yy — копировать текущую строку.
 • p — вставить после курсора.
 • P — вставить перед курсором.

Режим команд:
 • Чтобы войти в режим команд, нажмите : в нормальном режиме.
 • :w — сохранить файл.
 • :q — выйти из Vim.
 • :wq — сохранить и выйти.
 • :q! — выйти без сохранения.
 • :e имя_файла — открыть другой файл.

Настройка Vim
 1. Файл конфигурации:
Конфигурация Vim хранится в файле ~/.vimrc. Для изменения настроек откройте файл:

vim ~/.vimrc


 2. Пример базовой конфигурации:
Пример простого файла ~/.vimrc:

set number            " Включить нумерацию строк
set relativenumber    " Относительная нумерация
set tabstop=4         " Установить размер табуляции в 4 пробела
set shiftwidth=4      " Установить размер сдвига в 4 пробела
set expandtab         " Заменить табуляции на пробелы
syntax enable         " Включить подсветку синтаксиса
set background=dark   " Тема для тёмного фона

Работа с плагинами в Vim

Для установки плагинов удобно использовать менеджер плагинов, такой как Vundle или vim-plug.

Установка vim-plug:
 1. Установите vim-plug:

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
  https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim


 2. Откройте файл ~/.vimrc и добавьте плагин:

call plug#begin('~/.vim/plugged')

" Пример плагина: синтаксическая подсветка для Python
Plug 'vim-python/python-syntax'

call plug#end()


 3. После добавления плагинов в конфигурацию, откройте Vim и выполните команду:

:PlugInstall
